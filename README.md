# Smart Component Exporter

## Overview
The Smart Component Exporter is a SketchUp plugin designed to simplify the exporting of components to various file formats. This plugin enhances user productivity by providing an easy-to-use interface and robust features.

## Features
- Export components to multiple formats (e.g., .skp, .dae, .fbx).
- Batch export options for multiple components.
- User-friendly interface with drag and drop functionality.
- Options to customize export settings for different formats.

## Installation
To install the Smart Component Exporter:
1. Download the latest release from the [releases page](https://github.com/ozansert58/Sketchup/releases).
2. Open SketchUp.
3. Go to `Extension` > `Install Extension...`.
4. Select the downloaded file and click `Open`.

## Usage
1. Open the plugin from the `Extensions` menu.
2. Select the components you wish to export.
3. Choose the desired export format and settings.
4. Click `Export` to save the files.
